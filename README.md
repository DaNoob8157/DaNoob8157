### [Boat Man Shooters](https://github.com/DaNoob8157/Boat-Man-Shooters)
This is a online multiple player game where the intention is to shoot other boats and to stay alive. It is a constant lobby in which there can be a winner at any given time, given who is surviving.
- **Language:** Python
- **Status:** Active Development  
- [Website](https://github.com/DaNoob8157/Boat-Man-Shooters#:~:text=who%20is%20surviving.-,DaNoob8157.github.io/Boat%2DMan%2DShooters/,-Resources)

### [Apple Music: Cracked Edition](https://github.com/DaNoob8157/Apple-Music-Cracked-Edition)
A lightweight, web-based alternative to the native Apple Music app for macOS.
- **Language:** Swift
- **Status:** Active Development
- [Releases](https://github.com/DaNoob8157/Apple-Music-Cracked-Edition/releases)

### [Wordle Python Edition](https://github.com/DaNoob8157/Wordle-Python-Editon)
